Crea una aplicación web moderna y de alta gama en Replit. Utiliza React y Vite para la interfaz de usuario. Conecta la aplicación a un CMS sin cabeza (Headless CMS) para gestionar el contenido de forma dinámica. Elige Sanity.io o Strapi para el CMS. La arquitectura del proyecto debe estar preparada para ser desplegada en Netlify.



El diseño y la experiencia de usuario deben estar profundamente inspirados en el sitio web de Elie Saab (eliesaab.com). Implementa una estética minimalista y elegante, con un fuerte enfoque en la narrativa visual.



**Requisitos Específicos:**



1.  **Home Page:** Una página de inicio con una experiencia visual inmersiva, utilizando videos de fondo o imágenes de alta resolución en pantalla completa. La navegación debe ser sutil y minimalista, con un énfasis en el contenido.



2.  **Sección de "Colecciones":**

    * **Página de Índice:** Una cuadrícula o galería de proyectos con efectos sutiles de `hover` y transiciones fluidas al hacer clic.

    * **Página de Detalle de coleccion:** Cada coleccion debe tener su propia página con una galería de imágenes y videos de alta resolución, un título destacado, una breve descripción, y una sección de "Story" o narrativa del proyecto.



3.  **Sección de "Acerca de" (Quién Soy):** Una página con una biografía elegante y una galería de fotos del autor. La tipografía debe ser limpia y legible.



4.  **Sección de "Contacto":** Un formulario de contacto simple y un listado de perfiles de redes sociales.



5.  **Tecnologías:**

    * **Frontend:** React (usando Vite para un desarrollo rápido).

    * **CMS:** Sanity.io o Strapi. Elige el que mejor se adapte a un esquema de contenido para proyectos visuales. El modelo de contenido debe incluir tipos para "Proyectos", "Galerías de Imágenes", "Videos" y "Biografía".

    * **Despliegue:** La estructura del proyecto debe ser compatible con Netlify.



6.  **Desafíos Adicionales:**

    * **Performance:** Asegura que el sitio cargue rápidamente, optimizando la carga de imágenes y videos (lazy loading).

    * **Animaciones:** Utiliza una biblioteca como `Framer Motion` para transiciones de página suaves y microinteracciones elegantes.

    * **Responsividad:** El diseño debe ser perfectamente adaptable a dispositivos móviles, manteniendo la elegancia visual y la experiencia de usuario.
